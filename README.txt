Install instructions

-Create a shortcut of the Mousjigger.exe file found in 'MouseJiggler/bin/debug/net 6.0/'

-Move the shortcut to the following folder location 
	-C:\Users{user}\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

*you can also access this folder this way
	-press cntl+r to open the run application
	-enter the text 'shell:startup' and click 'ok'
